Using the Piece Detail Page

The Piece Detail Page is what you will be
viewing most of the time during a lesson.
This page has a place called "Goals:" to put
instructions for the student.

There is a built-in metronome with a slider
as well as a text box to type values in.

There is an automatic timer that starts when
you go to this page.   This allows you to
time how long you are spending on a piece
during a lesson.  It also works for the student
by allowing them to monitor how much time
they spend practicing on each piece.

There is a built-in recorder which can be used
to record one recording per piece per lesson.
It is really meant as a scratch pad for audible
information you want to pass on to the student.
The recording can be sent via Bluetooth or
email.  Sending a recording by Bluetooth can
take a while but if you are patient it can be done.
